# About
Savvy adalah aplikasi kirim pesan antar pengguna. Aplikasi ini berjalan secara realtime menggunakan socket.io antara sisi server dan client. Aplikasi ini dibuat menggunakan React Native untuk sisi client nya. Untuk backend nya menggunakan Expressjs dengan bantuan deploy backend dari AWS. Backend : bit.ly/3dULXHW
